# Igreja Batista Supere — Formulário de Conexão

Formulário interativo e elegante para recepção de visitantes da Igreja Batista Supere. Sistema multi-step com design moderno e responsivo.

## 🎯 Features

- ✨ Design moderno e responsivo para todas as idades
- 📱 Adaptável para mobile, tablet e desktop
- 🎨 Tema customizável com variáveis CSS
- ✅ Validação de formulário em tempo real
- 🎭 Animações suaves e fluidas
- ♿ Acessibilidade otimizada
- 📍 Perguntas personalizadas para novos visitantes
- 📲 Integração com WhatsApp

## 📁 Estrutura do Projeto

## 🚀 Como usar

### Abrir localmente

1. Clone ou baixe o repositório
2. Abra `index.html` no seu navegador
3. Pronto! O formulário está funcional

### Colocar no ar

Faça upload dos arquivos para seu servidor web:
- Use GitHub Pages, Vercel, Netlify ou seu próprio servidor
- Mantenha a estrutura de pastas

## 🎨 Customização

### Cores

Edit `src/css/styles.css`:
```css
:root {
  --gold: #c9a84c;      /* Cor primária */
  --black: #050507;     /* Fundo escuro */
  --white: #f0ede8;     /* Texto */
}
```

### Textos

Edit `index.html` para alterar perguntas e textos.

### Logo

Substitua `src/img/supere.png` com sua imagem (preferência: círculo com fundo transparente).

## 📊 Dados coletados

O formulário coleta:
- Nome completo
- Idade e data de aniversário
- Como conheceu a igreja
- Se conhece alguém da comunidade
- Participação em grupo espiritual
- Preferência por notificações
- Contato WhatsApp
